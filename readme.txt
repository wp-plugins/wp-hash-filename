Rename upload filename by hash.

== Description ==

[Plugin homepage](http://xiedexu.cn/wp-hash-filename.htm) | [Plugin author](http://www.iopenv.com)

This plugin can rename your upload filename by hash, no matter what language files.

== Installation ==

1. Upload `wp-hash-filename` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
