Rename upload filename by hash.HASH重命名上传文件名。

== Description ==

[Plugin homepage](http://xiedexu.cn/wp-hash-filename.htm) | [Plugin author](http://www.iopenv.com)

This plugin can rename your upload filename by hash, no matter what language files.这个插件可以HASH重命名上传文件，不管文件名名称，特别是我们经常遇到的中文乱码问题得以解决。

== Installation ==

1. Upload `wp-hash-filename` folder to the `/wp-content/plugins/` directory. 上传文件夹“wp-hash-filename”到“/wp-content/plugins/”目录。
2. Activate the plugin through the 'Plugins' menu in WordPress. 在仪表盘插件菜单中激活该插件。
